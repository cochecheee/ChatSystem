  <name>Task 3: GitHub Polling Service</name>
  <files>mcp/src/services/poller.py, mcp/src/main.py, mcp/.env.example</files>
  <action>
    - Tạo `mcp/src/services/poller.py` với class `GitHubPoller`:
      ```python
      class GitHubPoller:
          def __init__(self, github_client, processor, db_session_factory):
              self.interval = int(os.getenv("POLLING_INTERVAL_SECONDS", "300"))
              self.workflow_name = os.getenv("POLLING_WORKFLOW_NAME", "Security Scans")
              self.owner = os.getenv("GITHUB_OWNER")
              self.repo = os.getenv("GITHUB_REPO")

          async def start(self):
              while True:
                  await asyncio.sleep(self.interval)
                  await self._poll()

          async def _poll(self):
              runs = await self.github_client.get_workflow_runs(
                  self.owner, self.repo, self.workflow_name
              )
              last_id = await self._get_last_processed_run_id()
              for run in runs:
                  if run["id"] > last_id and run["conclusion"] == "completed":
                      await self.processor.process_run(run)
                      await self._save_last_processed_run_id(run["id"])
      ```
    - Extend `GitHubClient` với method `get_workflow_runs(owner, repo, workflow_name)`.
    - Thêm `Project` table field: `last_processed_run_id` (BigInt, nullable).
    - Khởi động poller trong `mcp/src/main.py` lifespan event:
      ```python
      @asynccontextmanager
      async def lifespan(app):
          asyncio.create_task(poller.start())
          yield
      ```
    - Thêm env vars vào `.env.example`:
      ```
      GITHUB_OWNER=your-org
      GITHUB_REPO=java-target-repo
      POLLING_INTERVAL_SECONDS=300
      POLLING_WORKFLOW_NAME=Security Scans
      ```
