<task type="auto">
  <name>Task 1: Khởi tạo security-scans.yml với Semgrep và CodeQL</name>
  <files>[java-repo]/.github/workflows/security-scans.yml</files>
  <action>
    Tạo `.github/workflows/security-scans.yml` trong Java repo:
    - Triggers: `push` và `pull_request` trên `main` branch.
    - Permissions tại workflow level:
      ```yaml
      permissions:
        contents: read
        security-events: write
        pull-requests: write
        actions: read
      ```
    - Job `semgrep`:
      - `uses: returntocorp/semgrep-action@v1`
      - `config: "p/default p/security-audit p/java p/javascript"`
      - `generateSarif: "1"`
      - `continue-on-error: true`
    - Job `codeql`:
      - `uses: github/codeql-action/init@v3` với `languages: java, javascript` và `java-version: '21'`
      - Build step: `./gradlew build -x test`
      - `uses: github/codeql-action/analyze@v3`
      - `continue-on-error: true`
    - Set job timeout: `timeout-minutes: 30` cho mỗi job.
  </action>
  <verify>
    <automated>gh workflow view security-scans.yml</automated>
  </verify>
  <done>Workflow file tạo xong với Semgrep và CodeQL jobs.</done>
</task>

<task type="auto">
  <name>Task 2: Cấu hình ESLint cho Thymeleaf Static Assets</name>
  <files>
    [java-repo]/src/main/resources/static/package.json,
    [java-repo]/.github/workflows/security-scans.yml
  </files>
  <action>
    - Trong thư mục `src/main/resources/static/` của Java repo, tạo `package.json` với dev dependencies:
      ```json
      {
        "devDependencies": {
          "eslint": "^9.x",
          "@microsoft/eslint-formatter-sarif": "^3.1.0",
          "eslint-plugin-security": "^4.0.0"
        }
      }
      ```
    - Tạo `eslint.config.js` trong cùng thư mục để include `eslint-plugin-security`.
    - Thêm job `eslint` vào workflow:
      ```yaml
      eslint:
        runs-on: ubuntu-latest
        timeout-minutes: 15
        steps:
          - uses: actions/checkout@v4
          - uses: actions/setup-node@v4
            with:
              node-version: '20'
          - name: Install ESLint
            working-directory: src/main/resources/static
            run: npm install
          - name: Run ESLint
            working-directory: src/main/resources/static
            run: npx eslint js/ --format @microsoft/eslint-formatter-sarif --output-file ../../../eslint-results.sarif
            continue-on-error: true
      ```
  </action>
  <verify>
    <automated>cd src/main/resources/static && npm list @microsoft/eslint-formatter-sarif eslint-plugin-security</automated>
  </verify>
  <done>ESLint job cấu hình xong, scan được Thymeleaf JS/TS assets.</done>
</task>
