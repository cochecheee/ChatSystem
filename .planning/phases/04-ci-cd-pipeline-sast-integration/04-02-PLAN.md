<task type="auto">
  <name>Task 1: Cấu hình SpotBugs trong build.gradle</name>
  <files>[java-repo]/build.gradle</files>
  <action>
    Cập nhật `build.gradle` trong Java repo để thêm SpotBugs plugin:
    ```gradle
    plugins {
        // ... existing plugins
        id 'com.github.spotbugs' version '6.0.0'
    }

    spotbugs {
        toolVersion = '4.8.3'
        ignoreFailures = true   // Không fail Gradle build, chỉ report
        effort = 'max'
        reportLevel = 'low'     // Report tất cả severity levels
    }

    spotbugsMain {
        reports {
            sarif {
                required = true
                outputLocation = layout.buildDirectory.file('reports/spotbugs/main.sarif')
            }
            html { required = false }
            xml  { required = false }
        }
    }
    ```
    - Thêm job `spotbugs` vào workflow:
      ```yaml
      spotbugs:
        runs-on: ubuntu-latest
        timeout-minutes: 30
        steps:
          - uses: actions/checkout@v4
          - uses: actions/setup-java@v4
            with:
              java-version: '21'
              distribution: 'temurin'
          - name: Setup Gradle
            uses: gradle/actions/setup-gradle@v3
          - name: Build and Run SpotBugs
            run: ./gradlew classes spotbugsMain
            continue-on-error: true
      ```
  </action>
  <verify>
    <automated>./gradlew spotbugsMain && ls build/reports/spotbugs/main.sarif</automated>
  </verify>
  <done>SpotBugs generate được SARIF file tại build/reports/spotbugs/main.sarif.</done>
</task>

<task type="auto">
  <name>Task 2: Tích hợp OWASP Dependency-Check</name>
  <files>[java-repo]/.github/workflows/security-scans.yml</files>
  <action>
    Thêm job `dependency-check` vào workflow:
    ```yaml
    dependency-check:
      runs-on: ubuntu-latest
      timeout-minutes: 30
      steps:
        - uses: actions/checkout@v4
        - name: Cache Dependency-Check DB
          uses: actions/cache@v4
          with:
            path: ~/.dependency-check
            key: dep-check-db-${{ hashFiles('build.gradle', '**/package-lock.json') }}
        - name: Run OWASP Dependency-Check
          uses: dependency-check/Dependency-Check_Action@v1.1.0
          with:
            project: 'java-target'
            path: '.'
            format: 'SARIF'
            out: 'dependency-check-results.sarif'
          continue-on-error: true
    ```
    - Cache database dựa trên hash của `build.gradle` và `package-lock.json` để tăng tốc.
  </action>
  <verify>
    <automated>gh workflow view security-scans.yml | grep "dependency-check"</automated>
  </verify>
  <done>Dependency-Check job scan cả Java và JS dependencies.</done>
</task>

<task type="auto">
  <name>Task 3: Unified Artifact Upload cho tất cả 5 jobs</name>
  <files>[java-repo]/.github/workflows/security-scans.yml</files>
  <action>
    Thêm upload step vào **mỗi** job (semgrep, codeql, eslint, spotbugs, dependency-check):
    ```yaml
    - name: Upload SARIF Results
      uses: actions/upload-artifact@v4
      if: always()   # Upload kể cả khi scan tìm thấy lỗ hổng
      with:
        name: security-results-${{ github.job }}
        path: "**/*.sarif"
        if-no-files-found: ignore
    ```
    - Dùng tên artifact khác nhau per job (`security-results-semgrep`, etc.) vì GitHub Actions không cho phép upload trùng tên trong cùng run.
    - Security gate job (plan 03) sẽ download tất cả và merge.
  </action>
  <verify>
    <automated>gh workflow view security-scans.yml | grep -c "upload-artifact"</automated>
  </verify>
  <done>Tất cả 5 jobs đều upload SARIF artifact sau khi scan xong.</done>
</task>
