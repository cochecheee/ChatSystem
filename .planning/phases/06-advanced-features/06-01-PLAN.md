  <files>mcp/src/api/chat.py, mcp/src/main.py</files>
  <action>
    Tạo `mcp/src/api/chat.py`:
    ```python
    router = APIRouter(prefix="/api/chat", tags=["chat"])

    # Role permission map
    COMMAND_ROLES = {
        "explain": ["developer", "security_lead", "admin"],
        "fix":     ["developer", "security_lead", "admin"],
        "report":  ["developer", "security_lead", "admin"],
        "scan":    ["security_lead", "admin"],
        "rerun":   ["security_lead", "admin"],
        "approve": ["security_lead", "admin"],
        "revoke":  ["security_lead", "admin"],
    }

    @router.post("/command", response_model=CommandResponse)
    async def handle_command(
        request: CommandRequest,
        current_user: User = Depends(get_current_user),  # JWT decode
        db: AsyncSession = Depends(get_db),
        background_tasks: BackgroundTasks = BackgroundTasks(),
    ):
        cmd = request.command.lstrip("/")
        allowed_roles = COMMAND_ROLES.get(cmd)
        if not allowed_roles:
            raise HTTPException(400, f"Unknown command: /{cmd}")
        if current_user.role not in allowed_roles:
            raise HTTPException(403, f"/{cmd} requires role: {allowed_roles}")
        return await command_service.handle(cmd, request, current_user, db, background_tasks)
    ```
    Register router trong `main.py`.
  </action>

    **`/explain` và `/fix`:**
    - Fetch finding từ DB theo `finding_id`
    - Gọi `LLMAnalysisService.analyze_vulnerability()`
    - Lưu kết quả vào DB, trả về `AnalysisOutput`

    **`/scan`:**
    - Gọi `GitHubClient.dispatch_workflow(owner, repo, workflow_name="Security Scans")`
    - Trả về `{"run_url": "..."}`

    **`/rerun`:**
    - Gọi `GitHubClient.rerun_workflow(run_id)`
    - Trả về `{"status": "triggered"}`

    **`/approve`:**
    ```python
    async def _handle_approve(finding_id, justification, user, db):
        finding = await db.get(Finding, finding_id)
        if not finding:
            raise HTTPException(404, "Finding not found")
        if finding.status == "APPROVED":
            raise HTTPException(409, "Finding đã được approve rồi")
        if finding.severity == "INFO":
            raise HTTPException(400, "INFO findings không cần approve")
        if not justification or len(justification) < 20:
            raise HTTPException(422, "Justification phải có ít nhất 20 ký tự")

        finding.status = "APPROVED"
        finding.justification = justification
        finding.approved_by = user.username
        finding.approved_at = datetime.utcnow()
        await db.commit()
    ```

    **`/revoke`:**
    - Logic tương tự approve nhưng:
      - Check `finding.status == "REVOKED"` → 409
      - Lưu vào `revoke_justification`, `revoked_by`, `revoked_at`
      - Set `finding.status = "REVOKED"`

    **`/report`:**
    - Gọi `ReportService.generate_html(findings)` — xem Task 4.
