  <files>mcp/src/services/llm/service.py</files>
  <action>
    - Tạo `mcp/src/services/llm/service.py`.
    - Implement class `LLMAnalysisService`:
      ```python
      class LLMAnalysisService:
          def __init__(self, client: GeminiClient):
              self.client = client
              self._semaphore = asyncio.Semaphore(3)  # max 3 concurrent calls

          async def analyze_vulnerability(self, request: AnalysisRequest) -> AnalysisOutput:
              async with self._semaphore:
                  prompt = build_prompt(request)
                  raw = await self.client.generate(
                      prompt=prompt,
                      system_instruction=SYSTEM_INSTRUCTION,
                      response_schema=AnalysisOutput
                  )
                  return AnalysisOutput.model_validate_json(raw)
      ```
    - Dùng `asyncio.Semaphore(3)` để giới hạn concurrent calls đến Gemini (tránh quota burst).
    - Log mỗi analysis request với `finding_id` và thời gian xử lý.
